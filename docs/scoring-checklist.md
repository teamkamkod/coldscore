# Cold Email Scoring Checklist

Score your outreach emails across 5 quantified dimensions.

## Quick Scoring Reference

| Dimension | Max | Score | What to Look For |
|-----------|-----|-------|-----------------|
| 🎯 Personalization | /25 | | Namedrop? Company reference? Recent event? |
| 🪝 Opening Hook | /25 | | Does first sentence make you read the second? |
| 💡 Value Proposition | /25 | | Clear, specific, quantified benefit? |
| 📞 Call to Action | /25 | | One clear, low-friction next step? |
| 📝 Conciseness | +5 | | Under 150 words? |

**Total: /100**

## Grade Guide

- **70-100** ✅ Solid — minor tweaks before sending
- **45-69** ⚠️ Needs work — several dimensions below 60%
- **0-44** ❌ Major rewrite — start over with the recipient in mind
